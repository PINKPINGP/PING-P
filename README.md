# PING-P

PING-P is an autonomous meme agent website built with React and Vite.

## Included in this package
- Updated landing page
- Whitepaper PDF (`public/whitepaper.pdf`)
- GitHub Pages workflow
- Relative-path Vite config for GitHub Pages

## Local development
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
```
